# Quoril

**Quoril is an AI-powered career intelligence platform that helps early-career professionals discover, evaluate, and rank job opportunities based on real-world signals—not just job titles.**

## 🧠 Overview

Traditional job platforms treat job search as a list.

Quoril treats it as a **decision system**.

It aggregates job data, enriches it with AI-generated insights, and ranks opportunities based on user intent, preferences, and likelihood of success.

## 🎯 Problem

Early-career candidates—especially in tech—face:

- Limited awareness of alternative roles (e.g., Solutions Engineer, Sales Engineer, Systems Engineer)
- Over-saturation of competitive roles (e.g., Software Engineer)
- Lack of structured insight into job difficulty, skill alignment, and hiring likelihood
- Generic job boards that don’t adapt to user intent

## 💡 Solution

Quoril provides:

- **AI-structured job insights**  
  Break down job descriptions into clear, structured data

- **Personalized ranking engine**  
  Rank jobs based on what the user actually cares about:
  - skill match
  - proximity
  - company size
  - likelihood of landing an interview

- **Intent-aware system**
  - Role-focused users get precision filtering
  - Exploratory users discover adjacent roles they may not know about

- **Career intelligence layer**
  Not just “find jobs” → but **understand your options**

## ⚙️ Core Features (MVP)

- 🔍 Job aggregation (scraped or seeded data)
- 🧠 AI-powered job summarization and structuring
- 📊 Dynamic job ranking based on user preferences
- 👤 User profile with weighted priorities
- 🧭 Intent modes:
  - Focused (no role suggestions)
  - Exploratory (adjacent role discovery)

## 🏗️ Tech Stack

- **Frontend:** Next.js (React)
- **Backend:** Next.js API Routes
- **Database:** PostgreSQL
- **AI Layer:** OpenAI / Claude API
- **Deployment:** Vercel
